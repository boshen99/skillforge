# Skill Factory

> **一句话**：给我一个意图，还你一个完整 Skill
> **七步元认知**：搜→分→叠→展→衡→验→成
> **中文版**：原理→PRINCIPLES.md，配方→RECIPES.md
> **English**: Principles→PRINCIPLES_en.md, Recipes→RECIPES_en.md

---

## 触发条件

```
当说出以下类型的意图时，进入七步推导：
  "我想做一个xxx"
  "能不能加一个xxx功能"
  "xxx应该怎么做"

Step 0（第零步）永远先走：
  搜"[你的意图] + Claude Skill" / "xxx workflow" / "xxx AI automation"
  搜到了 → 学习它的工具/流程/异常/阈值，再推导
  没搜到 → 确认空白，从零推导
```

## 七步推导总览

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
步骤   操作   问什么问题           输出什么
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
0      搜     网上有没有类似的？   学习工具/流程/异常/阈值
1      分     AI最容易做什么过头？  阴阳（做/不做）
2      叠     需要什么家伙？       四力（工具/任务/判断/门控）
3      展     这个Skill在哪？      八卦（输入/输出/边界/流向等）
4      衡     传导链断了吗？       五行（金→木→火→水→土）
5      验     有没有断点？         回头补（补到不断为止）
6      成     推导完成了吗？        完整Skill ✓
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 快速推导（5分钟版）

```
第零步：搜（1分钟）
  搜"_____ Claude skill" / "_____ workflow" / "_____ AI automation"
  搜到了？→ 学它的工具/流程/异常/阈值
  没搜到？→ 确认空白，从零推导

第一步：分阴阳（1分钟）
  AI最容易在___上做过头 → ✗ 禁止1
  ___属于上游/下游范围  → ✗ 禁止2
  ___必须留给人类判断   → ✗ 禁止3
  阳面：这个Skill要___   → 做1/做2/做3

第二步：叠四力（1分钟）
  太阳（工具）：___（工具）___（工具）___（工具）
  少阳（任务）：Step1 → Step2 → Step3
  少阴（判断）：判断1：___？  判断2：___？
  太阴（门控）：□ 完成标准1  □ 完成标准2

第三步：展八卦（1分钟，最少3个）
  乾（输入）：___     坤（输出）：___
  艮（边界）：___     巽（流向）：___

第四步：衡五行（30秒）
  金→木→火→水→土 哪一环断了？→ 回头补

第五步：验（30秒）
  木（方向）：扩展预设是否已定义？
  火（判断）：判断依据是否充分？判断是否过度僵化/松散？
  土（沉淀）：产出是否有冗余？
  金（结构）：边界是否过度僵化/松散？
  水（流动）：流动是否阻塞/过度发散？

第六步：成 ✓
  分✓ 叠✓ 展✓ 衡✓ 验✓ 成✓
```

## 质量检查（收口用）

```
□ 搜：搜过现有 Skill（搜了/没搜到/确认空白）
□ 分：禁止 >= 3 条
□ 叠：工具有 3-10 个，任务有序，判断有量化阈值
□ 展：至少填了乾（输入）+ 坤（输出）+ 艮（边界）
□ 衡：金→木→火→水→土 传导链不断
□ 验：木/火/土/金/水 五力各问过了，没有明显失衡
□ 成：分✓ 叠✓ 展✓ 衡✓ 验✓ 成✓
□ 扩展预设：扩展方向 + 扩展规则 + 扩展门控都有
```
